# 🌐 A股市场情绪分析Web面板 - 快速启动

## 一键启动

```bash
./start_web.sh
```

然后在浏览器中访问: **http://localhost:5000**

## ✨ 功能特点

- 🎯 **实时情绪分析**: 智能判断市场多空（-1到+1评分）
- 📈 **5大指数追踪**: 上证、深证、创业板、沪深300、中证500
- 🔍 **市场广度**: 涨跌统计、涨跌停实时监控
- 💡 **操作建议**: 根据情绪自动生成仓位建议
- 📊 **历史趋势**: 情绪得分时间序列图表
- 🔄 **自动刷新**: 每60秒自动更新数据
- 📱 **移动端**: 完美支持手机和平板访问

## 📊 情绪得分说明

| 得分 | 情绪 | 建议 |
|------|------|------|
| ≥ 0.6 | 🚀 强烈看多 | 仓位80-100% |
| 0.3~0.6 | 📈 看多 | 仓位60-70% |
| 0.1~0.3 | ↗️ 偏多 | 仓位30-50% |
| -0.1~0.1 | ➡️ 中性 | 仓位<30% |
| -0.3~-0.1 | ↘️ 偏空 | 仓位<20% |
| -0.6~-0.3 | 📉 看空 | 仓位<10% |
| ≤ -0.6 | 💥 强烈看空 | 完全空仓 |

## 📁 文件说明

- `web_server.py` - Flask Web服务器
- `templates/sentiment_dashboard.html` - 前端面板
- `market_sentiment_enhanced.py` - 情绪分析引擎
- `start_web.sh` - 启动脚本

## 🌐 访问地址

- **本地**: http://localhost:5000
- **局域网**: http://<你的IP>:5000

## 📡 API接口

```bash
# 获取情绪数据
curl http://localhost:5000/api/sentiment

# 强制刷新
curl http://localhost:5000/api/refresh

# 健康检查
curl http://localhost:5000/api/health
```

## 🛠️ 手动启动

```bash
source venv/bin/activate
python web_server.py
```

## 📱 移动端访问

1. 确保手机和电脑在同一WiFi
2. 查看电脑IP: `ifconfig` 或 `ipconfig`
3. 手机浏览器访问: `http://<电脑IP>:5000`

## 💡 使用建议

- **开盘前**: 查看当日市场情绪
- **盘中**: 实时监控情绪变化
- **收盘**: 查看全天趋势总结
